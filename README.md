Bergentemp
==========

App laget for bt.no.
Ressurser: R og Highcharts.
Den henter temperaturdata fra eklima målt i Bergen sentrum.
R-skriptet vasker dataene og produserer en JSON-fil.
Denne skubbes inn i en HTML med Highcharts i bunn, optimalisert for mobilvisning og desktopvisning i bt.nos CMS.
